macOS 快捷指令配置指引

由于 Apple Music API 在没有付费开发者账号时无法直接用本地网页控制，我们使用 macOS 自带的 快捷指令 (Shortcuts) 一键同步。这完全免费且非常安全。

1
创建快捷指令: 打开 Mac 上的“快捷指令” App，点击右上角新建，并重命名为 MusicCue (需与设置名称完全一致)。
2
配置流程节点 (添加以下 4 个动作):
  1. 添加 “拆分文本” (Split Text)：将 “快捷指令输入” (Shortcut Input) 以 “换行符” (New Lines) 拆分。
  2. 添加 “重复运行每个项目” (Repeat with Each)：循环遍历拆分后的文本。
  3. 在循环体内添加 “搜索 iTunes Store” (Search iTunes Store)： 将搜索项设为 “重复项目” (Repeat Item)，类别设为“歌曲”，限制改为 1，地区设为“美国”。
  4. 在循环体内添加 “添加至‘接着播放’” (Add to Playing Next)： 将 “iTunes 产品” (iTunes Products) 添加到“接着播放”的 “稍后” (Later)。
3
一键后台静默同步: 配置好后关闭快捷指令 App。点击下方“确认同步”按钮，系统会自动检测并调用，秒级同步推荐歌曲到你 Music 播放队列中！
小贴士: 制作此快捷指令后，系统将自动变为 **一键直接同步**，下次使用无需再阅读此弹窗教程。
